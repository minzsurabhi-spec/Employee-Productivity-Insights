# Employee-Productivity-Insights
Features an interactive Power BI dashboard designed to track and analyze workforce efficiency.

# Project Overview
​This project features an interactive Power BI dashboard designed to track and analyze workforce efficiency. It provides a visual breakdown of employee performance across various departments (Finance, HR, IT, Marketing, Operations, and Sales), helping managers identify high performers and areas for improvement.


​# Key Features & Insights
​**Performance Distribution** : A visual breakdown showing the ratio of 'Excellent' vs. 'Satisfactory' ratings (e.g., 33.3% Excellent).
​**Efficiency Tracking** : Calculated an Average Efficiency of 90% across the monitored tasks.
​**Workload Analysis** : Tracked Total Tasks Assigned (150) vs. Completed (135) to measure output.
​**Departmental Slicers** : Interactive filters to view specific data for different teams.
​

# Tools Used
​Power BI: Data visualization and dashboard design.
​Microsoft Excel: Data cleaning and initial structuring.
​DAX (Basic): Used for calculating measures like Average Efficiency and Task Completion rates.
